# NMTY
- Powered by Nuxt & Tailwind CSS
- Using Directus

## Setup
1. Clone the repository
2. Run `pnpm install`
3. Run `pnpm dev`

## Deploy
We deploy NMTY using Cloudflare Pages. [Read more](https://nuxt.com/deploy/cloudflare)
